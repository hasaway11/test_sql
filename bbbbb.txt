좋습니다. 'SID,SERIAL#' 자리에 들어갈 값을 확인하려면 아래 SQL을 실행해서 현재 락을 걸고 있는 세션의 SID와 SERIAL# 값을 알아내야 합니다.

✅ 1. 락 걸고 있는 세션 확인 SQL
sql
복사
편집
SELECT
  s.sid,
  s.serial#,
  s.username,
  s.status,
  o.object_name,
  s.program
FROM
  v$locked_object l
  JOIN dba_objects o ON l.object_id = o.object_id
  JOIN v$session s ON l.session_id = s.sid;
이 쿼리를 실행하면 다음과 같은 결과가 나옵니다:

SID	SERIAL#	USERNAME	OBJECT_NAME	STATUS	PROGRAM
123	45678	DEV_USER	MY_TABLE	ACTIVE	SQL Developer ...

✅ 2. 해당 세션 종료 명령
이제 아래처럼 작성해서 세션을 종료할 수 있습니다:

sql
복사
편집
ALTER SYSTEM KILL SESSION '123,45678' IMMEDIATE;
🔥 주의: 다른 개발자나 시스템에서 사용하는 세션이 아닐지 확인 후 종료하세요.
